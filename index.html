Projet : Site de Gestion de Stock et Point de Vente !

📅 Durée totale du projet : 4 semaines

🎯 Objectif : Développer une application mobile sous Ionic Angular permettant de résoudre et de factoriser des polynômes en utilisant différentes méthodes.

📌 Phase 1 : Planification et Analyse (Semaine 1)

  • ✅ Définition des fonctionnalités principales :
  • ✅ Étude de la base de données MySQL et conception du schéma
  • ✅ Création du cahier des charges et des maquettes (Figma)
    • Livrables : Cahier des charges détaillé, schéma de la base de données, wireframes du site

📌 Phase 2 : Développement du Backend (Semaines 2)

  • ✅ Mise en place de la base de données MySQL
  • ✅ Développement des modèles PHP (CRUD : Produits, Catégories, Sous-catégories,Marques)
  • ✅ Gestion des ventes : enregistrement et annulation
  • ✅ Création du système d'authentification des utilisateurs (login, logout, gestion des sessions)
    • Livrables : Base de données fonctionnelle, API PHP opérationnelle

📌 Phase 3 Développement du Frontend et Interface Utilisateur (Semaines 3)

  • ✅ Mise en place de Bootstrap pour le design
  • ✅ Création des pages principales :
  • ✅ Intégration AJAX pour des requêtes dynamiques sans rechargement de page
    • Livrables : Interface utilisateur fonctionnelle avec interaction fluide

📌 Phase 4 : Tests, Optimisation et Déploiement (Semaines 4)

  • ✅ Tests fonctionnels et correction des bugs
  • ✅ Sécurisation des données (protection contre les injections SQL, validation des entrées)
  • ✅ Optimisation des performances (indexation MySQL, mise en cache)
  • ✅ Hébergement du site sur un serveur web (en Local pour l'instant)
    • Livrables : Site web en production, documentation utilisateur et technique

🔍 Bilan du projet !

⏳ Durée totale : 4 semaines respectées

📲 Résultat final : Un site de gestion de stock et de point de vente intuitif et performant.

  • ✅ Ajouter un système de gestion des fournisseurs/li>
  • ✅ Génération de rapports PDF des ventes et stocks
